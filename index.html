We Connect The Dots Intermediate JavaScript Samples

This is the code repository used for the We Connect the Docs 20203 Level 2 JavaScript learning sessions.

There are several folders, in chronological order as presented.

  1. 01-hello-world—Basic "Hello World" in JavaScript
  2. 02-console-log—Logging to the JavaScript Console
  3. 03-a-proper-hello—Saying "Hello" using string concatenation
  4. 04-an-alternative-hello—Another way to concatenate strings to say "Hello"
  5. 05-computation—Doing math with JavaScript
  6. 06-decisions—Making decisions using JavaScript
  7. 07-longer-decisions—Making complex decisions using JavaScript
  8. 08-getting-elements—Using JavaScript to get elements in your HTML
  9. 09-changing-styles—Changing styles using JavaScript
  10. 10-reacting-to-a-click—Responding to a user click to do something
  11. 11-getting-input—Reading text fields so that the page is interactive
  12. 12-submitting-a-form—How to submit a form without reloading the page
  13. 13-arrays—A golf score card using arrays with score calculation
  14. 14-charting-arrays-of-objects—Uses a static data set to chart same data using arrays of objects
  15. 15-fetching-data—fetches data from OpenMeteo and charts the results in an interactive page. Illustrates using arrays and objects.
